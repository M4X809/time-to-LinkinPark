A React-based countdown timer app for tracking upcoming and past concert events. Built with TypeScript, Vite, and Tailwind CSS.

## Features

- **Countdown Timers**: Track time until upcoming concerts
- **Count-up Timers**: See how long it's been since past concerts
- **Person Filtering**: Filter events by specific people
- **Responsive Design**: Clean, dark-themed UI
- **PWA Ready**: Progressive Web App support

## Tech Stack

- React 19 + TypeScript
- Vite for fast development
- Tailwind CSS for styling
- React Router for navigation
- Date-fns for date handling

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Usage

- **Home Page**: View countdown timers for upcoming concerts
- **Past Page**: See count-up timers for past events
- **Person Select**: Filter events by specific attendees

The app automatically syncs timers and provides a smooth user experience for tracking your concert schedule.
